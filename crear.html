Factiun

1 · Datos base

El largo del tracker sale de los módulos/ala (28 → 64,7 m · 32 → 74 m). La zona UTM hace falta para situar el satélite cuando el archivo trae coordenadas UTM (DXF o CSV x,y). Si el CSV trae lat,lng, no hace falta.

2 · Implantación (seguidores)

Arrastra el CSV o DXF aquí, o pulsa para elegir
CSV: lat,lng o x,y (+ opc. id,tipo,ncu,gw) · DXF: bloques de tracker (INSERT) en UTM

¿Tienes un .dwg nativo? (no se lee en el navegador)

El DWG es binario propietario: lo procesa una GitHub Action con libredwg (lo que se usó con San José). Dos vías: a) exporta el DWG a DXF en tu CAD y súbelo aquí (instantáneo); b) súbeme el .dwg + estos datos base y te genero la planta y la dejo fija en la cartera.
